專利技術

光學感應感應器軌道系統，將原鋼琴觸鍵轉換成聲音來源

  1. 反光板型的光學非觸碰式感應器，能帶來百分之百原鋼琴的觸鍵感受，如同真實鋼琴般地重製琴音與敏銳度。
  2. 世界最薄的感應器軌道，所有鋼琴皆適用。
  3. 業界最高，每音符擁有9個層次與128個動態層級。
  4. 踏板光學非觸碰式感應器，能偵測踏板動作，並讓系統能呈現實際的鋼琴表現。

特色

  1. 靜音系統(靜音/鋼琴) 繞過鋼琴琴弦，透過耳機享受高解析度琴音。
  2. 128種聲音選擇音樂曲資料庫裡挑選。
  3. MIDI與adsilent裝置相容，如:電腦或其它MIDI樂器。
  4. 錄音對於您自己的表演，可進行監控、錄音、重播，演奏數據也能透過iTunes管理與分享。
  5. 內建節拍器功能依節拍器功能，彈奏正確節拍與速度。
